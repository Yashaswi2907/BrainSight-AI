# 🧠 BrainSight AI

An AI-powered Brain Tumor Detection System that analyzes MRI scans, predicts tumor types, generates Grad-CAM visualizations, and creates automated medical reports.

## 🚀 Live Demo

### Frontend
https://brain-sight-ai.vercel.app

### Backend API
https://brainsight-ai-1.onrender.com

### API Documentation
https://brainsight-ai-1.onrender.com/docs

---

## ✨ Features

- MRI Brain Scan Upload
- Brain Tumor Classification
- Multi-Class Prediction
- Grad-CAM Explainability Visualization
- Automated PDF Medical Report Generation
- Responsive UI
- Cloud Deployment
